# 🧠 Multiple Disease Prediction using Machine Learning Algorithms

## 📌 Project Overview
This project is a web-based machine learning system that predicts multiple diseases based on user input symptoms and health parameters.  
The system uses trained ML models to assist in early disease detection and improve healthcare awareness.

It is developed as a full-stack web application with a user-friendly interface and backend prediction engine.

---

## 🚀 Features
- 🔍 Predicts multiple diseases using ML models
- 🧾 User input-based prediction system
- 📊 Data-driven analysis using medical datasets
- 🖥️ Interactive web interface
- 👤 User registration and login system
- 📈 Accurate prediction using trained algorithms
